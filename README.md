## Descrição da solução :
```sh
 Guardámos os valores dos stats em listas onde podíamos ir buscálos utilizando o index do stat que pretendermos chamar.

 Utilizámos a "External library" "Python curses" para criar uma consola costumizada que permite selecionar opções utilizando as setas do teclado e mudar a cor do texto.

 Para desenhar a interface no ecrã utilizámos uma função "main" que lê o valor da variável "current_menu" e vai chamando as funções que desenham o menu correspondente.
```
